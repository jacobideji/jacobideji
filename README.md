#### What I'm building

* **AI IR Overlay™** is a practical incident response framework for teams deploying AI agents in production. It builds on NIST SP 800-61 r3 and aligns with NIST AI RMF 1.0, NIST CSF 2.0, OWASP Top 10 for Agentic Applications 2026, ISO/IEC 42001, and the EU AI Act. The framework is open source under Apache 2.0. Version v0.32.0 ships the full set of twenty-four playbooks, three standards crosswalks, JSON schemas with a strict validator, two reference implementations (an evidence exporter and a kill-switch demo), three QUICKSTART paths for CISOs, engineers, and startups, and a complete incident walkthrough you can follow end to end.

* **Field Manual for Cyber Incident Handling and Response: Prevent. Respond. Recover.** is a book I co-authored with Austin Weedor. It is available in paperback, Kindle, and Audible.

* **AI IR Overlay** is also a LinkedIn newsletter. Across twenty-four issues, I have written about agent identity risk, the evidence sources that are starting to matter, and the governance choices that authorized actors now face.
